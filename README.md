# HR-Analytics-Dashboard
Developed an interactive HR Analytics Dashboard in Power BI to analyze employee attrition patterns and workforce demographics using a dataset of 1,470 employees. The dashboard provides actionable insights into employee turnover, helping organizations identify key factors influencing attrition and improve retention strategies.

# Key Features
Calculated and visualized overall employee attrition rate (16.1%).
Analyzed attrition trends across multiple dimensions, including:
Age Group
Job Role
Salary Bracket
Gender
Years at Company (Tenure)
Created interactive filters and drill-down capabilities for detailed workforce analysis.
Built custom DAX measures and KPIs for HR performance monitoring.

# Key Insights
Identified higher attrition rates among employees in lower salary brackets.
Found the 26–35 age group to be the most affected by employee turnover.
Highlighted critical workforce segments requiring focused retention initiatives.
